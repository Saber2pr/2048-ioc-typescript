# 2048-ioc-typescript

> 2048 canvas game, use ioc model.

```bash
# from github
git clone https://github.com/Saber2pr/2048-ioc-typescript.git
```

---

## start

```bash
# install the typescript and webpack
npm install
```

```bash
# auto compile to commonjs
npm start

# auto compile to es5
npm run dev

```
